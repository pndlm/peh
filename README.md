# peh3 - PNDLM Environment Helper

Instructions WIP....

## Install the helper

```bash
GOPROXY=direct go install github.com/pndlm/peh/helper/peh@latest
```

## Add peh3 to the project

```bash
GOPROXY=direct go get github.com/pndlm/peh/peh3
```

## Build the helper

```bash
go build ./helper/peh
```
